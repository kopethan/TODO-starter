# TODO Product Spec Bundle

This bundle is a portable planning pack for future chats. It is **not** meant to be dropped directly into the main project repo.

It summarizes the current agreed product direction for:
- the public feed
- entities, reports, signals, and sources
- AI-assisted discovery and summarization
- information-sharing flows
- AI trend seeding with admin validation
- visual and interaction direction
- a suggested workflow for future implementation chats

## Core direction in one paragraph

TODO should be an **entity-first trust platform**. The feed should default to **Entities**, not a mixed stream of entities, reports, and signals. Each entity card can show counts and lightweight previews of related reports and signals. The real depth belongs on the **Entity detail page**, where users can understand the subject, switch between Overview / Reports / Signals / Sources, inspect grounded evidence, and use an AI assistant or information-sharing flow that stays structured and reviewable.
