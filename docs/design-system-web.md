# Web design system notes

## Direction
- monochrome-first
- smoky indigo shell theme
- flat design only
- black/white content surfaces with restrained color

## Shell principle
The shell carries the product identity. Content surfaces stay mostly neutral.

## Usage rules
- use color intentionally, not decoratively
- use borders more than shadows
- keep badges quiet
- keep primary buttons dark neutral instead of bright brand colors
- use semantic colors only for moderation, danger, and important notes
